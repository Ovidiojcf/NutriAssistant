Bem-vindo ao NutriAssistant! Como utilizar o sistema
  1. Cadastrar Paciente: Clique em Adicionar no menu inferior para registrar um novo paciente. Preencha todos os campos obrigatórios, incluindo alergias e comorbidades.
  2. Pesquisar Paciente: Use o botão Pesquisar para localizar pacientes já cadastrados. Utilize o campo de busca para filtrar pelo nome.
  3. Detalhar Paciente: Clique sobre um paciente na lista para ver detalhes, editar informações ou iniciar uma nova triagem.
  4. Realizar Triagem: No perfil do paciente, clique em Efetuar Triagem para preencher a ficha nutricional. Siga as etapas do formulário com atenção.
  5. Gerar PDF: Após preencher a triagem, clique em Gerar PDF para salvar e visualizar o documento. O PDF ficará disponível no histórico do paciente.
  6. Consultar Histórico: No perfil do paciente, acesse a lista de PDFs gerados para consultar triagens anteriores.
Atenção:
  • Preencha todos os campos obrigatórios para garantir a qualidade dos dados.
  • Revise as informações antes de salvar ou gerar o PDF, pois não é possível editar uma triagem já finalizada.
  • Proteja seus dados de acesso e não compartilhe sua senha.
  • Em caso de dúvidas, consulte o suporte ou o manual do sistema.
